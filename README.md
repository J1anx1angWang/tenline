# 十号线 · TENLINE

一款独立原创、离线优先的像素数字解谜游戏。拖动框选一个矩形；如果其中尚未消除的数字之和恰好为 10，这些数字就会被清除。

## 游戏模式

- 120 秒挑战：在倒计时结束前尽量消除更多数字，每消除一个数字得 1 分。
- 不限时解谜：没有倒计时，可以撤销、重做、重开同一棋盘或生成新谜题。

棋盘提供三种尺寸：8 × 14、9 × 15、10 × 16。清除后的空格留在原位，不会下落或补充；后续矩形可以跨过空格。

每局技能固定为：

- 提示 ×3：标出一个能消除较多数字的合法矩形。
- 重排 ×1：保留剩余数字，重新排列并验证至少存在一个合法组合。
- 单消 ×1：直接移除一个尚未消除的数字。

技能不会增加挑战模式分数。

## 操作

- 电脑：按住鼠标左键，从一个数字拖到另一个数字后松开。
- 手机和平板：手指按住一个数字，拖到目标格后松开。
- 框选时只显示矩形边框，不显示当前总和或答案颜色提示。

## 电脑离线使用

1. 下载并解压 `tenline-offline.zip`。
2. 双击目录中的 `index.html`。
3. 浏览器会直接启动游戏，不需要安装、登录或联网。

不同浏览器对本地文件存储的策略不同。即使最高分或解谜进度无法保存，核心游戏仍可正常运行。

## iPhone 安装与离线使用

1. 首次联网时，用 Safari 打开项目的 HTTPS 游戏地址。
2. 等页面完整加载后，点击“分享”。
3. 选择“添加到主屏幕”，确认以网页应用方式打开。
4. 从主屏幕启动一次。此后缓存有效时可以断网游玩。

如果版本刚更新，请联网打开一次，让浏览器替换旧缓存。

## Android 与电脑安装

在 Chrome、Edge 等支持网页应用的浏览器中打开 HTTPS 游戏地址，使用地址栏或浏览器菜单中的“安装”或“添加到主屏幕”。安装完成后可像普通应用一样启动。

## 本地开发

运行全部自动测试：

```bash
npm test
```

启动本地网页服务：

```bash
python3 -m http.server 4173
```

然后在浏览器访问 `http://127.0.0.1:4173/`。Service Worker 只会在 HTTP(S) 环境注册；直接打开本地文件时会自动跳过。

生成电脑离线包：

```bash
./scripts/make-release.sh
```

输出文件位于 `dist/tenline-offline.zip`。
